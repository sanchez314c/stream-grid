## [1.0.0] - 2025-08-23

### Added
- Initial release of StreamGRID
- Professional RTMP Multi-Stream Viewer
- Electron 28.x + React 18.x architecture
- Multiple grid layouts (1x1, 2x1, 3x1, 2x2, 3x3, 4x4)
- Stream controls (mute, volume, refresh, remove)
- Hardware acceleration support
- Stream library for saving frequently used streams
- Auto-reconnection on stream failure
- Professional dark theme UI
- SQLite database integration
- Real-time stream monitoring
- Performance optimization features

### Technical Features
- TypeScript support throughout
- Zustand state management
- React Query for server state
- HLS.js video streaming
- Winston logging
- Electron Store settings
- Tailwind CSS styling
- Vite build system
- Vitest testing framework
