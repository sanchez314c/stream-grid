# CHANGELOG

All notable changes to this project will be documented in this file.

## [Unreleased]

## [1.0.0] - 2025-09-01
### Added
- RTMP stream management system
- Flexible grid layouts (1x1 to 4x4)
- Individual stream controls (mute, volume, refresh)
- Hardware acceleration support
- Dark theme professional UI
- Stream library with SQLite storage
- Auto-reconnection on stream failure
- Multi-platform support (macOS, Windows, Linux)

### Changed
- Reorganized project structure to follow standards
- Updated build system for multi-platform distribution

- Basic grid layout support
- Initial UI implementation
